# Python Word Analyzer

## About this project

This is a simple Python project that reads a text file and analyzes its content. It provides basic information such as the number of words, lines, and characters, and also identifies the most frequently used words.

## What it does

* Counts the total number of words
* Counts the total number of lines
* Counts the total number of characters
* Finds the top 5 most used words

## Files in this project

* `word_analyzer.py` – the main program
* `sample.txt` – the text file used for testing
* `README.md` – project documentation

## How to run the program

1. Make sure Python is installed on your system
2. Open the project folder in terminal or VS Code
3. Run the following command:

python word_analyzer.py
